Offset cells in grid X

  • Place cells at a given grid line offset along the x axis.
.c-3.o-2.o-sm-3.o-md-4.o-lg-5

Offset cells in grid Y

  • Place cells at a given grid line offset along the y axis.
.c-3.o-2.o-sm-3.o-md-4.o-lg-5

Offset cells in grid XY

  • Place cells at a given grid line offset along the x or y axis, or both.
.c-3.o-2.o-sm-3.o-md-4.o-lg-5
